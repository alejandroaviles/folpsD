# CosmoSIS

{desi_likelihood}_priors.ini and {desi_likelihood}_values.ini are to be copied in a pipeline priors.ini and values.ini, which also include other (e.g. cosmological) parameters.

For BAO + SN inference, run:
```
cosmosis config_bao_all_sn.ini
```

For FS + BAO inference, run:
```
cosmosis config_fs_bao_all.ini
```
